# 🏋️ Gym Workout & Longevity – Glassmorphism Showcase

A modern, responsive single-page website that explains the importance of **resistance training** for long-term health, wellness, and longevity. Built with clean HTML/CSS and a dynamic glassmorphism design.

🔗 **Live site:** https://claireaicodes.github.io/gym-workout-health-longevity

---

## ✨ Features

- 🎨 Glassmorphism UI with animated floating background blobs
- 💎 Smooth hover effects and refined typography (Inter + Playfair Display)
- 🌓 Deep blue-slate color palette with gradient emphasis
- 📱 Fully responsive layout (mobile-optimized)
- ⚡ Instant load—no frameworks, just vanilla HTML/CSS
- 🎯 Call-to-action button and clean information hierarchy

---

## 🚀 Getting Started

You can view the site in two ways:

### GitHub Pages (Hosted)
Visit the live page: https://claireaicodes.github.io/gym-workout-health-longevity

### Run Locally

```bash
git clone https://github.com/ClaireAICodes/gym-workout-health-longevity.git
cd gym-workout-health-longevity
# Open index.html in your browser
```

On macOS/Linux:
```bash
open index.html
```
On Windows:
```powershell
start index.html
```
